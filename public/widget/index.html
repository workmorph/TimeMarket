🚀 TimeBid ウィジェット

📋 基本的な使用方法

あなたのウェブサイトに以下のコードを追加するだけで、TimeBidウィジェットを埋め込めます：

<!-- ウィジェットのスクリプトを読み込み -->
<script src="https://yourdomain.com/widget/timebid-widget.js"></script>

<!-- ウィジェットを表示する場所に配置 -->
<div data-timebid-widget></div>
🎯 自動初期化: data-timebid-widget 属性を持つ要素が自動的にウィジェットに変換されます

🎨 カスタマイズ例

より詳細なカスタマイズを行う場合：

<div id="timebid-custom-widget"></div>

<script>
document.addEventListener('DOMContentLoaded', function() {
    new TimeBidWidget({
        container: '#timebid-custom-widget',
        config: {
            theme: 'dark',
            primaryColor: '#ff6b6b',
            borderRadius: '12px',
            showLogo: true,
            customCSS: `
                .timebid-widget {
                    background: linear-gradient(45deg, #667eea 0%, #764ba2 100%);
                }
            `
        }
    });
});
</script>

🔧 設定オプション

オプション デフォルト 説明
theme 'light' 'light' または 'dark'
primaryColor '#0066cc' メインカラー（16進数）
borderRadius '8px' 角の丸み
showLogo true ロゴ表示の有無
customCSS '' 追加のCSSスタイル
